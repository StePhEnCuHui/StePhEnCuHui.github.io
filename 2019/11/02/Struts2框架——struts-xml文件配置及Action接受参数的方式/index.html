参数注入原理

​ 参数会自动注入是因为 struts-default.xml 中有两个拦截器在工作

​ 1.struts 的默认拦截器是由 defaultStack 决定的

​ 2.如果在 package 中不写拦截器，就是 struts-default 中的defaultStack 的拦截器在工作

​ 3.一旦在 package 中写了拦截器，struts-default 中的 defaultStack 的任何拦截器就都不工作了
